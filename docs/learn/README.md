## Как пользоваться

**Автору** (мутабельный режим):
- `quiz <topic>` — пройти тест, при провале LLM генерирует адресное пояснение в `<topic>/remediation/generated/`
- `study <topic>` — навигация по теме, статус прохождения
- `promote <qid>` — перенести пояснение из `generated/` в `static/`

**Гостю** (иммутабельный режим):
- Через `/learn` на сайте: главы, тесты, валидация ответов. Прогресс — в localStorage браузера.
- Через PDF: `bun run build:learn` → `learn-book.pdf` со всеми главами и дневником пробелов.
